## 解释口径

画像数量分布图回答“不同实验条件下，交易者行为画像组成如何变化”。图中使用普通柱状图而不是堆叠柱状图，避免在小子图中挤压标签。纵轴是各画像在该配置下的占比，源数据同时保留 agent 数量。

损益分布图回答“最终收益是否集中到少数钱包”。横轴为最终损益，纵轴为钱包密度，只使用平滑分布曲线，不再叠加柱状图。解释时重点看右尾是否拉长、是否出现少数高收益钱包，而不是只比较均值。

信念价格差距图回答“群体平均信念是否贴近模拟市场价格”。横轴为累积成交笔数，蓝线是平均信念，浅蓝区间是信念四分位区间，灰线是模拟市场价格。差距结论写入正文，不写在图内。

## 聚类优化说明

本轮聚类优化移除了 `mean_api_latency_ms`。该变量更容易反映 API 响应和运行环境差异，而不是交易者行为本身；保留它会让“信念锚定型”被推理耗时主导，导致该类只在极少数配置中出现。

重新拟合后，模型选择 4 类画像：激进套利型、信念锚定型、频繁挂撤型、信号反向型。K=4 的 bootstrap Jaccard 为 0.643，最小簇占比为 15%，比 K=5 更稳定。信念锚定型不再是孤立标签，而是在 38 个配置中出现，总计 236 个 agent。
