# v15 微观聚类与图表说明

## 聚类用途

微观画像来自每个 agent 的结构化行为特征与 reasoning 文本特征。聚类优化后移除了容易受 API 响应波动影响的 `mean_api_latency_ms`，重新拟合得到四类交易者画像：激进套利型、信念锚定型、频繁挂撤型、信号反向型。

当前图表只把聚类结果用于“画像数量分布”这一类图，不再额外输出特征指纹热图、收益矩阵或市场间持续性图。特征指纹仍可作为内部解释材料保留在表格和 `v15_behavioral_patterns.md` 中，但不作为正文主图。

## 聚类优化结果

当前模型选择 K=4。与 K=5 相比，K=4 的 bootstrap Jaccard 更高，最小簇占比也更稳定，因此更适合作为正文画像口径。

`表_4_2_行为模式_分布_全部.csv` 显示，“信念锚定型”在 38 个配置中出现，总计 236 个 agent。它不再是孤立标签，可以作为一种稳定画像讨论。
