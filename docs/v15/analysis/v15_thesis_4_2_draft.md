# 4.2 微观交易者行为分析草稿

本节从微观层面观察多智能体仿真是否形成可解释的交易者行为。为避免图表过散，正文只保留三类证据：画像数量分布、损益分布、信念价格差距。三类图表均按正常实验、规模扩展、决策轮数扩展和消融实验展开，使微观分析与前一节宏观实验保持一致。

## 4.2.1 正常实验下的微观表现

正常实验首先用于建立默认参数下的交易者画像结构。图 `4-2-1_微观_画像数量分布_正常实验` 显示，优化后的聚类模型将智能体划分为激进套利型、信念锚定型、频繁挂撤型和信号反向型四类。该划分移除了 API 推理耗时特征，因此更强调交易动作、信念变化、仓位反应和收益结果本身。

图 `4-2-1_微观_损益分布_正常实验` 用最终损益分布补充画像解释。若分布集中在 0 附近，说明大多数钱包收益有限；若出现右尾，则说明少数交易者在同一市场条件下获得更高收益。该图适合用于讨论微观收益分化，但不应单独证明策略优劣。

图 `4-2-1_微观_信念价格差距_正常实验` 展示群体平均信念与模拟市场价格的关系。该图的核心问题是：智能体的平均判断是否贴近市场价格，还是稳定高估或低估。若蓝线长期低于灰线，说明群体信念偏保守；若两者逐步靠近，说明交易过程和信念更新机制具有一定修正作用。

## 4.2.2 规模扩展下的微观变化

规模扩展实验观察智能体数量增加后，微观行为是否只是机械增加交易机会，还是改变交易者结构。图 `4-2-2_微观_画像数量分布_规模扩展` 用画像占比而不是总交易次数进行比较，因此可以避免把“人多导致交易多”误写成行为质量提升。

图 `4-2-2_微观_损益分布_规模扩展` 用最终损益分布观察规模放大后的收益分化。如果大规模条件下右尾变长，说明更多智能体提供了更大的分化空间，少数钱包可能从交易机会增加中获益；如果曲线整体仍集中在 0 附近，则说明规模增加未必带来明显个体收益优势。

图 `4-2-2_微观_信念价格差距_规模扩展` 用于判断群体规模是否改善信念与价格的一致性。若规模增大后平均信念更接近模拟市场价格，可以写作“规模扩展在一定程度上缓和群体认知与市场价格之间的偏离”；若差距仍然稳定存在，则说明单纯增加智能体数量不能消除模型先验偏差。

## 4.2.3 决策轮数扩展下的微观变化

决策轮数扩展实验关注更多交易轮次是否带来行为收敛。图 `4-2-3_微观_画像数量分布_决策轮数` 展示不同轮数下画像结构的变化。如果长轮数下画像更分散，说明智能体并不会自动走向单一理性策略，而是在持续观察、撤单、调价和重新判断中形成不同路径。

图 `4-2-3_微观_损益分布_决策轮数` 展示轮数增加后的损益分布变化。长轮数可能扩大右尾，也可能让错误信念被反复交易而扩大亏损，因此该图应与信念价格差距图一起解释，而不是简单写成“轮数越多越好”。

图 `4-2-3_微观_信念价格差距_决策轮数` 展示更多决策轮次是否让平均信念靠近模拟市场价格。若差距随轮数缩小，可以说明 UPDATE_BELIEF 的重复调用具有修正作用；若价格和信念同时朝错误方向移动，则说明更多轮次也可能放大初始误判。

## 4.2.4 消融实验下的微观变化

消融实验按画像分布、思考模式和信念更新工具三个模块展开。图 `4-2-4_微观_画像数量分布_消融实验` 观察模块开关是否改变交易者画像组成；图 `4-2-4_微观_损益分布_消融实验` 观察模块变化是否改变最终损益分布；图 `4-2-4_微观_信念价格差距_消融实验` 观察模块变化是否改变群体信念与模拟市场价格之间的距离。

其中，信念更新工具是最需要重点解释的模块。如果关闭 UPDATE_BELIEF 后画像分布变得单一、损益分布收缩或信念价格差距扩大，可以说明信念更新并不只是形式步骤，而是影响微观行为多样性和市场认知修正的重要机制。

## 4.2.5 小结

三类图表共同回答微观层面的研究问题：画像数量分布说明“谁在交易”，损益分布说明“交易结果如何分化”，信念价格差距说明“交易者认知是否靠近市场”。在写作上，应避免把总成交增加直接解释为行为改善，而应围绕画像结构、损益尾部和信念价格距离三个维度展开。
