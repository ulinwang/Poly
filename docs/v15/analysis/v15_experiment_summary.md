# v15 实验结果分析摘要

本轮实验使用“先更新 belief、再执行交易动作”的两阶段智能体框架。每个实验只运行 1 个 seed，重点是为论文第 4 章提供可解释的价格路径、交易者画像、损益分布与信念价格关系。

## 价格含义与判断标准

本文统一使用 YES 价格作为价格序列。YES 价格接近 1.00 表示市场更接近 YES 结算，YES 价格接近 0.00 表示市场更接近 NO 结算。因此，真实结局为 YES 的市场中，模拟价格上升并靠近 1.00 是方向正确；真实结局为 NO 的市场中，模拟价格下降并靠近 0.00 是方向正确。

## 4.1 宏观市场表现

宏观图表用于回答默认参数、智能体规模、决策轮数和模块消融如何影响模拟市场价格。解释时重点看价格漂移方向、终点距离真实结局的远近、路径波动和真实市场价格路径之间的关系。

规模实验和轮数实验不能只讨论总成交笔数，因为智能体数量或决策轮数增加本身就会提高潜在交易机会。更合适的解释是：在相同市场下，价格路径是否更接近真实方向，标准化成交指标是否变化，路径波动是否被放大。

## 4.2 微观交易者行为表现

微观章节只保留三类图表，每一类都分别对应正常实验、规模扩展、决策轮数扩展和消融实验。

第一类是画像数量分布图。它用于说明不同实验条件下四类交易者画像的数量占比如何变化。聚类优化后移除了 API 推理耗时特征，避免把运行环境差异误识别为行为模式；重新拟合后得到激进套利型、信念锚定型、频繁挂撤型和信号反向型四类画像。

第二类是损益分布图。它以最终损益为横轴、钱包密度为纵轴，只呈现平滑分布曲线。分析时重点看分布是否出现右尾拉长、少数高收益钱包或更宽的离散区间，用于讨论微观收益是否向少数交易者集中。

第三类是信念价格差距图。它以累积成交笔数为横轴，同时展示群体平均信念、信念四分位区间和模拟市场价格。分析时重点看平均信念是否持续低于或高于市场价格，以及规模扩展、轮数扩展和模块消融是否缩小这种差距。

## 4.3 活跃市场验证

活跃市场验证用于说明系统在市场关闭前的活跃阶段可以形成模拟价格路径，并在市场关闭后用最终结局进行事后验证。该部分不应被写成预测准确率证明，而应写成“活跃市场情境下的模拟有效性检查”。
